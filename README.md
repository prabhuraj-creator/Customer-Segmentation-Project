# Customer Segmentation Project

## Objective
This project performs customer segmentation using Machine Learning.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Algorithm Used
K-Means Clustering

## Features
- Customer clustering
- Data visualization
- Customer behavior analysis

## Output
- Elbow Method Graph
- Segmentation Visualization
- Cluster Summary

## Conclusion
Customer segmentation helps businesses identify target customers and improve marketing strategies.
